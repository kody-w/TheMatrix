## Animation Timeline

```

─────────────────
0s:   Hero metrics appear (fade in)
0.5s: Primary number animates in (counter)
1.0s: Secondary metrics cascade in
1.5s: Agents visualization appears
2.0s: Agent progress bars start animating
3.0s: Cost meter starts incrementing
3.5s: Time comparison appears
4.0s: All metrics stabilized

INTERACTION ANIMATIONS:
─────────────────────
Hover card:       0.3s lift + shadow
Click to expand:  0.4s smooth open
Scroll trigger:   Animate in as visible
Scenario change:  0.8s number update
Tab switch:       0.3s fade transition
```

## Mobile Responsiveness

```
DESKTOP (1200px+)           TABLET (768px)          MOBILE (375px)
──────────────────────      ──────────────          ─────────────
5 cards in row      →       3 cards in row    →     1 card (stacked)
Hero metrics 60px   →       Hero metrics 48px →     Hero metrics 36px
2-column layout     →       Single column     →     Single column
Full animations     →       Reduced motion    →     Simplified anim
Side-by-side comp   →       Stacked comp      →     Tabbed interface
```
