## Usage

All games are standalone HTML applications:

1. Open any `.html` file in a modern browser
2. Read on-screen controls/instructions
3. Grant permissions if requested (gamepad, etc.)
4. Enjoy!

## Game Controls

### Common Keyboard Controls
- **WASD** or **Arrow Keys**: Movement
- **Space**: Jump/Action
- **ESC**: Pause/Menu
- **Mouse**: Look/Aim
- **Click**: Select/Fire

### Touch Controls
- **Swipe**: Move/Navigate
- **Tap**: Select/Action
- **Pinch**: Zoom
- **Long Press**: Context menu

### Gamepad Support
Many games support standard gamepads:
- **Left Stick**: Move
- **Right Stick**: Look/Aim
- **A Button**: Jump/Select
- **B Button**: Back/Cancel
- **Start**: Pause
