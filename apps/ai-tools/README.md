# AI Tools Collection

This directory contains 16 specialized AI development and deployment tools built for The Matrix framework.

## Applications

### Agent Development & Management

**agent-browser.html**
- Browse and explore AI agent configurations
- Interactive agent specification viewer
- Capability mapping and discovery

**claude-subagents-tutorial.html**
- Step-by-step guide to Claude Code subagents
- Hands-on examples and exercises
- Best practices for agent orchestration

**claude-subagents-tutorial-continued.html**
- Advanced subagent patterns
- Complex orchestration scenarios
- Production deployment guidance

**agent-deployment-prototype.html**
- Prototype AI agent deployment workflows
- Test deployment configurations
- Validate agent specifications

### AI Model & Prompt Management

**aplAI.html**
- AI prompt library and management
- Version control for prompts
- Performance tracking and optimization

**aplai-watch-app.html**
- Real-time prompt performance monitoring
- A/B testing interface
- Analytics dashboard

**mcp-registry.html**
- Model Context Protocol (MCP) server registry
- Browse available MCP integrations
- Configuration management

### Integration & Deployment

**Copilot_Companion.html**
- Microsoft 365 Copilot development companion
- Extension testing and validation
- Integration workflow builder

**copilot-agent-store.html**
- Marketplace for Copilot agents and extensions
- Browse, search, and deploy agents
- Community sharing platform

**consolidated-ai-tools.html**
- Unified dashboard for all AI tools
- Centralized configuration management
- Cross-tool workflows

### Voice & Conversation

**complete-voice-assistant.html**
- Full-featured voice AI assistant
- Speech-to-text and text-to-speech
- Conversational AI workflows

**workshop.html**
- AI workshop and experimentation environment
- Test AI concepts rapidly
- Prototype new agent capabilities

**wristAI.html**
- Wearable AI interface (watch/wristband)
- Quick AI interactions
- Context-aware assistance

### Simulation & Testing

**ai-simulation-sales-demo.html**
- Simulate AI-powered sales scenarios
- Demo environment for stakeholders
- Customizable workflows

## Features

- Voice recognition APIs
- Custom webhook support

## Usage

All tools are standalone HTML applications that run directly in your browser:

1. Open any `.html` file in a modern web browser
2. Grant necessary permissions (microphone for voice apps, etc.)
3. Follow the on-screen instructions
4. Configure integrations as needed

## Development

### Technology Stack
- Pure HTML5, CSS3, JavaScript (ES6+)
- No build process required
- Web Components for modularity
- Service Workers for offline support

### Adding New Tools
1. Create new `.html` file following naming convention
2. Include standard header with purpose comment
3. Implement responsive design patterns
4. Add tool to this README
5. Update main app gallery

## Best Practices

### For Agent Development
- Start with tutorials before building custom agents
- Use agent-browser to explore existing patterns
- Test deployments with the prototype tool
- Monitor performance with watch app

### For Prompt Engineering
- Manage prompts centrally in aplAI
- Version control all prompt changes
- A/B test with aplai-watch-app
- Share successful prompts with team

### For Integration
- Review MCP registry for existing integrations
- Use consolidated-ai-tools for orchestration
- Test with workshop before production
- Document custom integrations
